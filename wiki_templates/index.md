# PipecatAssistant Wiki

Private notes and memory for the assistant. See `schema.md` for the rules
that govern edits.

## Layout

- `personas/` — voice + prompt + tool allowlist per persona_id
- `devices/` — device → persona mapping + device notes
- `people/` — per-user facts (preferences, recurring topics, relationships)
- `daily/` — append-only turn log (`YYYY-MM-DD.md`), one block per turn
- `log.md` — one-line-per-turn audit trail
- `schema.md` — librarian's rulebook for editing pages
